### Images

For inline images, use standard markdown syntax:

```
![Alt text for image](../assets/images/stories/triberg/image-name.jpg)
```

Images are stored in `assets/images/stories/{post-slug}/` directory.

### YouTube Videos

Embed YouTube videos using this custom syntax:

```
[youtube: VIDEO_ID]
```

Example:
```
[youtube: dQw4w9WgXcQ]
```

The renderer will convert this to an embedded iframe. Video ID is the part after `v=` in YouTube URLs.

## Best Practices

1. Keep paragraphs concise (2-3 sentences max)
2. Use descriptive alt text for all images
3. Break up text with images every 2-3 paragraphs
4. Use blockquotes for tips and important information
5. Embed YouTube videos for tutorials or scenery
6. End with a call-to-action or summary
